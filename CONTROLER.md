# Controler

## 概要

本アプリの操作を行うメインのコントローラーUIです。

構成は以下の通りとする。
1. 複数の Menu item から構成する。
2. Menu item は横一列に配置する。
3. アプリウインドウの下部中央に表示する。

## Menu item の種類

Menu item は以下二種類のUIから構成される。
- 円
- 正方形(角丸)

### 円

吹き出しを持つ UI です。

クリックすることで、吹き出しの表示/非表示をコントロールします。

### 正方形(角丸)

吹き出しを持たない UI です。

選択状態を持ち、クリックすることで選択状態になります。

### 吹き出し

円 UI をクリックすることで表示される UI です。

吹き出しの表示位置は、円UIのX軸の中心と吹き出しのX軸の中心を同じ位置になるように表示します。

## Menu item の構成

Menu item の構成は 左から、ログ照会、データ取得、ユーザー、歯車、+ (プラス)、ボード とする。

### ログ照会

(1) Menu item の種類
円

(2) メニューの概要
出力ログを照会する。

### データ取得

(1) Menu item の種類
円

(2) メニューの概要
ボードのデータ取得を行う

### ユーザー

(1) Menu item の種類
円

(2) メニューの概要
Slackアカウントの管理

### 歯車

(1) Menu item の種類
円

(2) メニューの概要
現在選択れているボードの設定、ボードの削除

### + (プラス)

(1) Menu item の種類
円

(2) メニューの概要
ボードの追加

### ボード

(1) Menu item の種類
正方形(角丸)

(2) メニューの概要
ボードの条件で slack チャンネル、メッセージを表示する。
